# U2I Reimagined

A modern, high-performance web application for **Univers Inox (U2I)**, specializing in industrial piping and orbital welding solutions. This project reimagines the U2I digital presence with a premium, responsive, and interactive experience.

## 🚀 Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Meta-framework**: [TanStack Start](https://tanstack.com/start) & [TanStack Router](https://tanstack.com/router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/) (Radix UI)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Data Visualization**: [Recharts](https://recharts.org/)
- **Package Manager**: [Bun](https://bun.sh/)

## ✨ Key Features

- **Interactive Hero Section**: Dynamic sliders showcasing industrial expertise.
- **Service Showcases**: Dedicated sections for Pharmaceutical, Food, Chemical, and Cosmetic industries.
- **Equipment Gallery**: High-quality visualization of industrial machinery and orbital welding equipment.
- **Partner & Certifications**: Grid display of official partners (AXXAIR, Sanofi, etc.) and industry certifications.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.
- **Video Integration**: Immersive background videos and presentations.

## 🛠️ Getting Started

### Prerequisites

Ensure you have [Bun](https://bun.sh/) installed on your machine.

### Installation

```bash
# Clone the repository
git clone https://github.com/mohamedgueddes/u2i-reimagined.git

# Navigate to the project directory
cd u2i-reimagined

# Install dependencies
bun install
```

### Development

Run the development server:

```bash
bun dev
```

The app will be available at `http://localhost:3000`.

### Build

To create a production build:

```bash
bun build
```

## 📂 Project Structure

- `src/assets`: Images, videos, and icons used across the site.
- `src/components`: Reusable UI components (Shadcn UI and custom components).
- `src/routes`: File-based routing using TanStack Router.
- `src/hooks`: Custom React hooks for mobile detection and other logic.
- `src/lib`: Utility functions and configuration.
